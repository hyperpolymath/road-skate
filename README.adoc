A retro driving game written in link:https://github.com/hyperpolymath/affinescript[AffineScript].

In spirit of Accolade's _Test Drive III: The Passion_ (1990) — perspective-projected
road rendering, centripetal-force steering model, and the kind of physics that makes
a car feel like it has weight.

toc::[]

== Status

Prototype. The pieces that exist:

* `road_render*.affine` — perspective-projection road renderer (multiple iterations;
  current target is `road_render.affine`).
* `physics_example.affine` — centripetal-force steering / car-state physics.
* `game/` — minimal browser host (`index.html`, `main.as`).

Not yet game-shaped: no input handling, no track definition, no audio. The
renderer + physics are the working substrate.

== Build

You'll need the AffineScript compiler
(link:https://github.com/hyperpolymath/affinescript[hyperpolymath/affinescript])
installed.

[source,bash]
----
just check     # type-check
just build     # compile to WASM
just run       # exercise physics_example.affine
----

The `Justfile` is the entry point.
